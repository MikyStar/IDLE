# IDLE

## Technos

### Client

- React
- TypeScript
- CSS' FlexBox & GridLayout system
- Docker

### Server

- NodeJS
- TypeScript
- MongoDB
- GraphQL
- Docker

## Running it

`You'll need the .env files obviously need commited but also Docker and docker-compose`

```sh
# To run the server
cd server && docker-compose up

# To run the client
docker build -t reactts . && docker run -it reactts
```
